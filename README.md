# Habit Tracker CLI

A pure Python command-line application to track and analyze your habits. This project uses SQLite for storage and provides analytics functionality using functional programming paradigms.

## Prerequisites

- Python 3.7 or higher
- No external libraries are required (use a standard library only).

## Installation

1. Unzip the project folder.
2. Open a terminal/command prompt.
3. Navigate to the project root directory.
